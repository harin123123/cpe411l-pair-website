SoleStyle Website

SoleStyle is a responsive, green-themed shoe shop website featuring a home page, about, contact, FAQ, and user authentication (login & sign up). The site includes styled cards, forms, buttons, and smooth animations for a modern look.

The project follows a strict GitHub workflow: all changes are made through feature/* branches, merged via pull requests, and reviewed by a partner before merging to main.
